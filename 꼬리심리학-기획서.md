## 2. 왜 꼬리심리학인가?

### 시장 기회

- 국내 반려동물 양육 가구 약 602만 (2024 기준), 매년 증가세
- MBTI, 색깔 테스트 등 성격 유형 콘텐츠의 바이럴 파워는 이미 검증됨
- 반려동물 + 성격 테스트 조합은 아직 국내에서 지배적인 서비스가 없음
- "우리 아이 성격이 뭐야?"는 반려인이라면 누구나 궁금해하는 질문

### 네이밍 전략

**꼬리심리학**을 메인 브랜드로, 강아지/고양이를 완전히 분리된 파트로 운영:

- 메인 브랜드: **꼬리심리학** — 신뢰감 + 호기심 유발
- 강아지 파트: **멍BTI** — 독립 진단 시스템 + 전용 질문셋 + 전용 유형 캐릭터
- 고양이 파트: **냥BTI** — 독립 진단 시스템 + 전용 질문셋 + 전용 유형 캐릭터
- SNS 해시태그: `#꼬리심리학 #멍BTI #냥BTI #내강아지유형 #내고양이유형`

> 강아지와 고양이는 행동 체계가 근본적으로 다르기 때문에, 질문/유형/AI 분석 모델을 각각 독립적으로 설계한다.

## 3. 성격 유형 시스템 설계

### 3-1. 진단 축 (4가지 차원)

MBTI처럼 4가지 축의 조합으로 성격 유형을 만들되, 강아지/고양이 각각에 맞게 독립 설계:

**🐶 멍BTI 진단 축**

| 축 | 양극단 | 설명 |
|----|--------|------|
| **에너지** | 🔥 활활 (A) ↔ 😴 느긋 (C) | 활동량, 흥분도, 산책 반응 |
| **사교성** | 🫂 다가감 (S) ↔ 🐾 마이웨이 (M) | 사람/다른 강아지에 대한 친화력 |
| **감수성** | 💧 섬세 (D) ↔ 🪨 담대 (B) | 소리, 환경 변화, 분리 불안 반응 |
| **호기심** | 🔍 탐험가 (E) ↔ 🏠 안전제일 (H) | 새로운 장소/물건에 대한 탐색 행동 |

**🐱 냥BTI 진단 축**

| 축 | 양극단 | 설명 |
|----|--------|------|
| **에너지** | ⚡ 사냥본능 (A) ↔ 💤 선잠 (C) | 놀이 빈도, 야간 활동량 |
| **사교성** | 🫂 집사러브 (S) ↔ 👑 독립왕 (M) | 사람 접촉 선호도, 무릎 위 빈도 |
| **감수성** | 💧 예민냥 (D) ↔ 🪨 대범냥 (B) | 낯선 손님, 환경 변화, 소음 반응 |
| **호기심** | 🔍 탐험냥 (E) ↔ 🏠 영역냥 (H) | 새 물건/상자에 대한 반응, 영역 행동 |

### 3-2. 유형 코드 예시

각 파트 4축 × 2선택 = 파트별 **16가지 유형**, 총 **32가지 유형**

**🐶 멍BTI 유형 (일부)**

| 유형 코드 | 캐릭터 이름 | 한 줄 설명 |
|-----------|-------------|------------|
| ASDE | 🌪️ 태풍이 | 활발하고 사교적이고 섬세하고 호기심 폭발 |
| ASBH | 🦁 사자왕 | 에너지 넘치고 당당하고 겁 없는 리더 |
| ASBE | 🎉 파티피플 | 어디서든 인기쟁이, 모든 강아지와 친구 |
| CMBH | 🧸 집순이 | 느긋하고 독립적이고 자기 자리를 좋아함 |

**🐱 냥BTI 유형 (일부)**

| 유형 코드 | 캐릭터 이름 | 한 줄 설명 |
|-----------|-------------|------------|
| ASDE | 🌊 호기심폭발냥 | 활발하고 집사 좋아하고 섬세한 탐험가 |
| CMDE | 🦉 현자냥 | 조용하지만 관찰력 좋고 섬세한 지식인 |
| CMDH | 👑 도도냥 | 조용하고 독립적이고 자기만의 세계가 확실 |
| ASBH | 🐆 야생냥 | 사냥본능 강하고 대범하고 영역 의식 확실 |

### 3-3. AI 기반 진단 시스템

> 꼬리심리학의 핵심 차별점: AI가 단순 설문을 넘어 실제 행동 데이터를 분석

**진단 파이프라인**

```
[입력] → [AI 분석] → [유형 산출] → [결과 생성]
```

**Stage 1: 설문 + AI 해석**

- 보호자가 15~20개 질문에 응답 (강아지/고양이 완전 분리 질문셋)
- AI(LLM)가 단순 점수 합산이 아닌, 응답 패턴을 종합 분석
- 모순된 응답 감지 → 추가 질문으로 정확도 향상
- 자연어 응답 옵션 지원: "우리 아이는 이런 행동을 해요" 자유 입력 → AI 해석

**Stage 2: 이미지/영상 AI 분석**

- 반려동물 사진 업로드 → 표정, 자세, 귀/꼬리 위치 분석
- 영상 업로드 → 행동 패턴 추출 (움직임, 반응 속도, 상호작용)
- Computer Vision 모델이 비언어적 행동 신호를 수치화

**Stage 3: 하이브리드 종합 진단**

- 설문 결과 + AI 행동 분석 교차 검증
- 신뢰도 점수 제공 ("이 진단의 정확도: 87%")
- AI가 유형별 맞춤 설명을 자연어로 생성 (매번 다른 표현)

**강아지 질문 예시 (멍BTI)**

- "산책 중 낯선 강아지를 만나면?" → 4지선다
- "현관 벨이 울리면 반응이?" → 4지선다
- "새로운 장난감을 주면?" → 4지선다
- 자유 입력: "우리 아이만의 특이한 행동이 있다면?"

**고양이 질문 예시 (냥BTI)**

- "상자를 놔두면 반응이?" → 4지선다
- "손님이 오면 어디로 가나요?" → 4지선다
- "레이저 포인터를 켜면?" → 4지선다
- 자유 입력: "우리 고양이만의 독특한 루틴이 있다면?"

## 6. 기술 스택

### 아키텍처 개요

```
[사용자] → [Cloudflare CDN/DNS] → [AWS 서버]
                                      ├── Next.js (프론트엔드)
                                      ├── FastAPI (백엔드 + AI 파이프라인)
                                      ├── PostgreSQL (DB)
                                      └── AI 모델 서빙
```

### 상세 스택

| 영역 | 기술 | 선정 이유 |
|------|------|-----------|
| **프론트엔드** | Next.js + TypeScript | SSR/SEO 유리, 기존 경험 활용 |
| **스타일링** | Tailwind CSS + Framer Motion | 빠른 UI 개발 + 부드러운 애니메이션 |
| **백엔드** | FastAPI (Python) | AI 파이프라인과 동일 언어, 비동기 고성능, 자동 API 문서화 |
| **AI — LLM** | OpenAI API / Claude API | 설문 해석, 자유 입력 분석, 맞춤 결과 텍스트 생성 |
| **AI — Vision** | YOLOv8 + OpenCV | 반려동물 자세/표정/행동 분석 (2차) |
| **AI — 추론** | FastAPI 내장 + 모델 서빙 | 경량 모델 직접 서빙, 대형 모델은 API 호출 |
| **DB** | PostgreSQL (AWS RDS 또는 자체 호스팅) | 안정성, 확장성, JSON 지원 |
| **캐시** | Redis | 세션 관리, AI 응답 캐싱, 결과 임시 저장 |
| **스토리지** | AWS S3 | 반려동물 사진/영상, 결과 카드 이미지 저장 |
| **배포 — 서버** | AWS EC2 / ECS | 유연한 스케일링, AI 모델 GPU 인스턴스 대응 |
| **배포 — CDN** | Cloudflare | CDN, DDoS 방어, DNS 관리, SSL |
| **배포 — 프론트** | Cloudflare Pages 또는 AWS 서빙 | Next.js 정적 + SSR 하이브리드 배포 |
| **분석** | GA4 + Mixpanel | 사용자 행동 추적, 바이럴 계수 측정 |

### AI 파이프라인 상세

```
[설문 응답 + 자유 입력]
    ↓
[FastAPI — 전처리]
  ├── 응답 정규화
  ├── 모순 감지 로직
  └── 자유 입력 텍스트 추출
    ↓
[LLM 분석 엔진]
  ├── 프롬프트 엔지니어링으로 4축 점수 산출
  ├── 자유 입력 해석 → 보조 성격 지표
  └── 유형 코드 결정 (ASDE, CMBH 등)
    ↓
[결과 생성 엔진]
  ├── LLM이 유형 + 개별 응답 기반 맞춤 설명 생성
  ├── 궁합 분석 텍스트 생성
  └── 양육 팁 생성
    ↓
[결과 저장 + 카드 렌더링]
```
